HTML Tags Research Task

Below is a list of 30 common HTML tags and their primary use cases in web development.

  1. <!DOCTYPE html> - Defines that this document is an HTML5 document.
  2. <html> - The root element of an HTML page.
  3. <head> - Contains meta information about the HTML page.
  4. <title> - Specifies a title for the HTML page (shown in the browser tab).
  5. <body> - Defines the document's body, containing all visible content.
  6. <h1> to <h6> - Defines HTML headings (h1 is the most important).
  7. <p> - Defines a paragraph of text.
  8. <br> - Inserts a single line break.
  9. <hr> - Defines a thematic change in the content (horizontal line).
  10. <nav> - Defines a set of navigation links.
  11. <ul> - Defines an unordered (bulleted) list.
  12. <ol> - Defines an ordered (numbered) list.
  13. <li> - Defines a list item within a list.
  14. <div> - Defines a division or a section in an HTML document.
  15. <span> - A container used to mark up a part of a text or document.
  16. <a> - Defines a hyperlink to another page or location.
  17. <img> - Used to embed an image in an HTML page.
  18. <main> - Specifies the main content of a document.
  19. <section> - Defines a section in a document.
  20. <article> - Defines an independent, self-contained piece of content.
  21. <header> - Defines a header for a document or section.
  22. <footer> - Defines a footer for a document or section.
  23. <table> - Defines a table for displaying data.
  24. <tr> - Defines a row in an HTML table.
  25. <td> - Defines a standard cell in an HTML table.
  26. <th> - Defines a header cell in an HTML table.
  27. <strong> - Defines important text (usually rendered in bold).
  28. <em> - Defines emphasized text (usually rendere d in italic).
  29. <button> - Defines a clickable button.
  30. <style> - Used to define internal CSS styles for a document.
